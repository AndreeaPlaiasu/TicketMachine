# Ticket Machine – Member A

Kotlin console application for COM534 Group Project (Solent University).

## Description
This project simulates a simple train ticket machine system.

Users can:
- View available destinations
- Choose between single or return tickets
- Insert money
- Receive a printed ticket in the console

## Files
- Destination.kt – Stores destination data
- Ticket.kt – Defines ticket properties
- TicketMachine.kt – Handles ticket selling and prices
- UserInterface.kt – Manages user input/output
- Main.kt – Starts the program

## How to Run
1. Download the repository.
2. Open it in IntelliJ IDEA.
3. Run `Main.kt`.
